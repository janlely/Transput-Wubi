# Transput

一个支持AI翻译的输入法，目前只支持五笔，基础框架fork自[Typut](https://github.com/ensan-hcl/Typut)


## Working Environment

Checked in March 2024.
* macOS 14.3
* Swift 5.10
* Xcode 15.3

## 使用指南
![image](./show.gif)

## 本地安装

* build
```bash

xcodebuild

cp -r build/Release/Transput.app ~/Library/Input\ Methods/
```

* 注销当前用户

* 在系统输入法设置中添加Transput
